# How?

# Run!
* `gulp test` tests with [Mocha](http://mochajs.org/)
* `gulp build` build app bundle with [Webpack](http://webpack.github.io/)
* `gulp start` start a [Node.js](https://nodejs.org/) [Express](http://expressjs.com/) web server
